# C - Sorting Algorithms & Big O Notation Project

## Description
This is a team project that is part of the one year intensive software engineering training at ALX. This project aims at enabling trainees gain an understanding of the types of sorting algorithms and when to select an algorithm that best suits a particular project. Some key points learned from this project are:

- Knowledge of at least four different sorting algorithms
- Understanding of Big O notation
- How to evaluate the time complexity of an algorithm
- How to select the best sorting algorithm for a given input
- Understanding of a stable sorting algorithm

## Usuage
- Files were created using vim
- All files were compiled on `Ubuntu 20.04 LTS` using `gcc`, with the options: `-Wall -Werror -Wextra -pedantic -std=gnu89`
